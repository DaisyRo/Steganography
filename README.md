# Digital Watermarking Toolkit
## LSB & Adaptive Steganography for Grayscale Images

Python-реализация цифровых водяных знаков для 8-битных полутоновых изображений. 

---

## Задание 2: Цифровые водяные знаки

### Реализованные методы:

1. **LSB с секретным ключом**
   - Встраивание в младшие битовые плоскости
   - Псевдослучайная перестановка позиций по ключу (SHA-256)
   - Объём внедрения: ≥50% ёмкости контейнера

2. **Адаптивное внедрение (Вариант 1)**
   - Адаптация по локальному градиенту (оператор Собеля)
   - Приоритет текстурным зонам и границам объектов
   - Избегание однородных областей

### Дополнительные функции:
- Проверка целостности: заголовок `WTR1` + SHA-256
- Пакетная обработка трёх наборов данных (BOSSbase, Medical, Other)
- Автоматический расчёт метрик качества (PSNR, MSE, SSIM)
- Доверительные интервалы (α=0.05) для статистического анализа
## Использование

### Пакетная обработка всех изображений

Запустите скрипт для внедрения водяного знака во все изображения из наборов `data/`:

```bash
python3 code/batch_process.py \
  --out results \
  --logo code/logo.png \
  --k 1 \
  --key secret \
  --capacity 0.5 \
  --window 5 \
  --shuffle
---
```
